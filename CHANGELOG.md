# 更新日志 / Changelog

## v0.4.0 (2026-06-27)

### 新增功能 / New Features

- **标题栏重新设计** — 参考 Apostrophe 风格，左侧"打开(O)"/"保存(S)"按钮，中间显示文件名，右侧工具按钮
- **标题栏下拉菜单** — "打开"按钮下拉显示最近文件列表，"保存"按钮下拉显示"另存为"和"新建文件"
- **首选项对话框** — 重新设计为 Apostrophe 风格的手风琴式设置面板
- **检查拼写** — 启用浏览器原生拼写检查功能
- **自动隐藏标题栏** — 输入时自动隐藏标题栏和状态栏，鼠标移到顶部区域可恢复
- **大号文本模式** — 减小边界宽度并增大文字大小
- **Markdown 输入格式选择** — 支持 Pandoc's Markdown / CommonMark / GFM / MultiMarkdown / 纯文本
- **搜索按钮** — 标题栏新增搜索图标，快速打开查找替换
- **大纲按钮** — 标题栏新增大纲图标，快速切换大纲面板
- **新建文件按钮** — 标题栏新增新建文件图标

### 功能优化 / Improvements

- **大纲导航** — 点击大纲标题现在可以在所有编辑模式（源码/分屏/预览）下滚动到对应位置
- **查找替换** — 现在在所有编辑模式下均可使用（之前仅源码模式）
- **工具栏图标** — 优化工具栏按钮图标显示
- **状态栏简化** — 精简状态栏信息，更接近 Apostrophe 的简洁风格
- **菜单面板** — 采用手风琴式交互，点击一级分类展开二级菜单

### 问题修复 / Bug Fixes

- 修复标题栏拖拽区域与按钮点击冲突的问题
- 修复自动隐藏标题栏后无法恢复显示的问题
- 修复大纲点击在预览模式下无法滚动的问题
